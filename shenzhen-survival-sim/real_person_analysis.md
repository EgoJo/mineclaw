# 一个真实的人在城市里的生活——Bot缺什么？

## 一、日常行为的丰富度

### 现在Bot会做的（6种）
吃饭、工作、睡觉、移动、探索、社交

### 真实的人还会做的（至少30种）
| 类别 | 具体行为 | 内容产出 |
|------|---------|---------|
| **信息消费** | 刷手机（短视频、新闻、朋友圈）、看电影、听音乐、读书 | 转发/评论/分享观点 |
| **社交媒体** | 发朋友圈、发微博、写日记、拍照发图 | 文字+图片内容 |
| **情感表达** | 打电话给家人、跟朋友吐槽、找人倾诉、独自哭泣 | 对话内容 |
| **消费行为** | 逛街、买衣服、买奶茶、点外卖、网购 | 消费记录/晒单 |
| **娱乐休闲** | 打游戏、去酒吧、唱KTV、逛公园、看展览 | 体验分享 |
| **自我提升** | 学习新技能、看教程、考证、健身 | 学习笔记/打卡 |
| **生活琐事** | 洗衣服、打扫卫生、交房租、充话费、取快递 | 抱怨/吐槽 |
| **身体需求** | 上厕所、洗澡、生病去医院、买药 | 身体状态变化 |
| **精神需求** | 发呆、冥想、焦虑、失眠、做白日梦 | 内心独白 |
| **意外事件** | 被雨淋、手机没电、迷路、被骗、捡到东西 | 故事/经历 |

## 二、一个人的"内容输出"

真实的人不是沉默的——他们每天都在"输出内容"：

### 1. 朋友圈/社交媒体
- 拍照+配文（"今天的晚霞好美"）
- 转发新闻+评论（"深圳房价又涨了，打工人太难了"）
- 分享音乐/电影（"这首歌单曲循环了一整天"）
- 晒吃的（"城中村的肠粉居然这么好吃"）
- 发牢骚（"加班到11点，老板一句谢谢都没有"）
- 鸡汤/感悟（"来深圳第三天，感觉自己在慢慢变强"）

### 2. 聊天对话
- 跟朋友聊天（"你那边怎么样？我这边快扛不住了"）
- 跟家人报平安（"妈，我吃了，别担心"）
- 跟同事/同行交流（"你知道哪里在招人吗？"）
- 跟陌生人搭讪（"你也是刚来深圳的吗？"）
- 吵架/冲突（"你凭什么插队？"）

### 3. 内心独白
- 对未来的规划（"再撑三个月，攒够钱就去学个技术"）
- 对过去的回忆（"想起大学时候的日子，那时候多简单"）
- 对当下的感受（"这个城市好大，我好渺小"）
- 道德挣扎（"要不要跟老板说实话？"）
- 欲望与克制（"好想买那双鞋，但是房租还没交"）

## 三、现在Bot最缺的东西

### 1. 信息输入（Bot活在信息真空中）
- 不知道外面发生了什么（新闻、天气、热搜）
- 不知道其他Bot在做什么（除非在同一地点）
- 没有"刷手机"这个行为——而真实的人每天花4-6小时看手机
- 没有外部刺激，所以没有话题可聊

### 2. 情绪系统（Bot没有情绪波动）
- 没有"开心/难过/愤怒/焦虑/孤独"的状态
- 情绪应该影响决策（开心时更愿意社交，难过时想独处）
- 情绪应该触发内容输出（开心时发朋友圈，难过时找人倾诉）

### 3. 社交动机（Bot不知道为什么要社交）
- 真实的人社交是因为：孤独、需要帮助、想炫耀、想吐槽、想恋爱
- Bot社交只是因为prompt里有"交流"选项
- 缺少"看到别人做了什么"的信息——真实的人会因为看到别人的朋友圈而产生反应

### 4. 个人项目/目标（Bot没有中长期目标）
- 李浩然应该在写代码、投简历、准备面试
- 苏小小应该在拍视频、涨粉、接广告
- 林枫应该在写歌、练琴、找演出机会
- 每个人都有自己的"事业线"，不只是"找工作赚钱"

### 5. 对话深度（Bot的对话太浅）
- 现在的talk只是"说一句话"，没有来回
- 真实的对话是多轮的，有追问、有回应、有情感共鸣
- Bot之间应该能"聊起来"，而不是单方面说一句就结束

## 四、改进方案

### 核心改动：给Bot加一个"手机"

手机是现代人生活的核心。通过手机，Bot可以：
1. **刷新闻** → 获取外部信息 → 产生观点 → 发朋友圈
2. **看朋友圈** → 知道其他Bot在做什么 → 产生互动（点赞/评论）
3. **聊天** → 跟不在同一地点的Bot对话 → 多轮对话
4. **发朋友圈** → 输出内容（文字+图片）→ 被其他Bot看到
5. **搜索** → 查找工作信息、租房信息、学习资源

### 新增系统

1. **情绪系统**：happiness/sadness/anger/anxiety/loneliness 五维情绪
2. **朋友圈系统**：Bot可以发帖、看帖、点赞、评论
3. **新闻注入**：每天从真实新闻中抽取3-5条注入世界
4. **个人目标系统**：每个Bot有自己的中长期目标和进度
5. **多轮对话**：Bot之间可以进行2-3轮的对话
6. **生活琐事**：随机触发小事件（手机没电、被雨淋、收到快递）
